Weather Risk API

Mountain-aware risk forecasting powered by Open-Meteo

A single JSON endpoint that fetches hourly data from up to four weather models (ECMWF, GFS, GEM, ICON) and returns a structured risk forecast aimed at outdoor and alpine users. Every response includes a 24-hour guide brief, a 10-day daily breakdown, and multi-model confidence scoring.

No API key required. Open-Meteo is free for non-commercial use. Rate-limit calls on your end — the server caches responses for 10 minutes within a 1 km radius.

Quick Start

Run the server

# requires Go 1.21+
git clone https://github.com/you/weather-risk-api
cd weather-risk-api
go run .
# → listening on :8080

Make a request

curl "http://localhost:8080/forecast?lat=41.33&lon=19.82"

Risk Categories

Each category is independently rated LOW MODERATE HIGH. The daily overall risk_level is escalated to HIGH if any individual category is HIGH, or if three or more are MODERATE (compound hazard).

WIND

Based on 700 hPa ridge wind (~3000 m) and 10 m gusts.

PRECIP

Prioritises peak hourly rate over total accumulation.

VISIBILITY

Minimum surface visibility in metres.

THUNDER

CAPE J/kg — tuned for Balkan afternoon convection.

FREEZING

Freezing precipitation type codes or freezing level ≤2200 m.

THERMAL

Wind chill using the Environment Canada formula.

Weather Models

All data is sourced from Open-Meteo. The four models are fetched concurrently. ECMWF is required; the others are best-effort.

ModelRoleFields fetched
ECMWF IFS Primary — all risk calculations use this 26 hourly fields including pressure-level wind, temp, geopotential, CAPE, visibility, precipitation type
GFS Spread model temperature_2m, wind_speed_10m, precipitation
GEM (Canadian) Spread model same as GFS
ICON (DWD) Spread model same as GFS

Confidence Scoring

Confidence reflects model agreement across ECMWF, GFS, GEM, and ICON for the first 24-hour horizon. Spread is measured as max − min of each model's 24-hour average.

LevelWind spreadPrecip spreadTemp spread
HIGH< 6 km/h< 1.5 mm/h< 2 °C
MEDIUM6 – 12 km/h1.5 – 3 mm/h2 – 4 °C
LOW≥ 12 km/h≥ 3 mm/h≥ 4 °C

If GFS, GEM, and ICON all fail, confidence is LOW with detail "Single-source".

Spatial Cache

The server keeps an in-memory list of recent forecasts. On each request the Haversine distance to every cached entry is checked. A cache hit is returned if a live entry exists within 1 km of the requested coordinates.

PropertyValue
TTL10 minutes
Radius1 km (Haversine)
EvictionExpired entries are dropped on each cachePut call
X-Cache headerHIT if served from cache, MISS if fresh fetch
